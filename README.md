## 📖 About

**SignTalk** is a Flutter-based mobile application that enables real-time sign language communication, built with a strong focus on accessibility for deaf and hard-of-hearing users. The app bridges communication gaps by offering multiple modalities — sign, text, and voice — so users can interact in whichever way suits them best, ensuring inclusive and seamless communication.

---

## ✨ Features

- 🔤 Real-time sign language communication via chat
- ⌨️ Text and voice input options
- 👤 User authentication and profile management
- 💬 Messaging system with real-time data sync
- 📊 Sign language chart reference
- 🚫 Contact management and blocking features
- ⚙️ Settings and preferences customization
- 📱 Comprehensive UI supporting multiple screen types

## 🎯 Project Focus

This capstone project was developed with a strong emphasis on **accessibility**, specifically designed to bridge communication gaps for deaf and hard-of-hearing users. By supporting multiple communication methods — sign, text, and voice — SignTalk ensures inclusive and seamless interaction between all users, regardless of their preferred mode of communication.
